# github.com/kimchanhyung98/dotfiles

Chanhyung's dotfiles, managed with [`chezmoi`](https://www.chezmoi.io/).

Install them with:

```console
$ chezmoi init kimchanhyung98
```

Or use the bootstrap script:

```console
$ curl -fsSL https://raw.githubusercontent.com/kimchanhyung98/dotfiles/main/install.sh | bash
```
